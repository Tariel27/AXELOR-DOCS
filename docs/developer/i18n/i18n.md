В Axelor есть метод интернационализации (i18n) '{{ 'Example' | t}}'. 

Схема использования в примере:

<html>
  <p>{{ 'PIN' | t}}: {{pin}}</p>
  <p>{{ 'Download Date' | t}}: {{dateOfDownload}}</p>
